# The Rep Studio

A studio hub + a set of standalone, hand-built demo websites for **gyms, studios and fitness & wellness brands** — the Fitness vertical of the BuildspaceLabs studio program (parent: Vruoom). Sibling to [The Pass Studio](https://buildpacelabs.github.io/the-pass-studio/) (F&B) and Atelier Travel Studio.

Every demo site is deliberately its **own** design world — palette, type, layout, motion, voice — so the set reads as a range of real independent businesses, not one template recoloured. Plain HTML, CSS and vanilla JavaScript. No build step, no framework.

## Live

- **Hub:** https://buildpacelabs.github.io/the-rep-studio/
- **Each demo:** https://buildpacelabs.github.io/the-rep-studio/&lt;slug&gt;/

## Structure

```
the-rep-studio/
  index.html          # the studio hub (generated by build_hub.py)
  sitemap.xml         # generated by build_hub.py
  robots.txt
  projects.json       # the source list of demo brands (drives the hub + sitemap)
  build_hub.py        # regenerates index.html + sitemap.xml from projects.json
  screenshot.js       # Playwright hub-card screenshots
  assets/shots/       # real screenshots of each deployed demo
  <slug>/             # index.html, styles.css, script.js, contact.html, thank-you.html
```
