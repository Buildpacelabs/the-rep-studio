The short version

Built on empty roads and full hearts

It started with four of us and a thermos of filter coffee.

In 2019 we were tired of running alone — dodging traffic, quitting at 3km, wondering if we were doing it right. So we picked a lamppost near Sankey Tank, set an alarm for 4:45, and told a few friends to show up. Eleven did.

Six years on, Tarmac is a crew of over four hundred runners who believe the same thing: the road is easier when someone's beside you. We pace the nervous first-timer and the sub-3 marathoner in the same breath, because pace is personal but the start line is shared.

No membership needed to try — just lace up and turn up.
